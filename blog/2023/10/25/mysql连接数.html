    MySQL连接数

    0. Linux 查看pid下mysql的连接数¶

    lsof -p pid | grep mysql | wc -l

    1. mysql最大连接数¶

    mysql> show variables like '%max_connections%';
+-----------------+-------+
| Variable_name   | Value |
+-----------------+-------+
| max_connections | 3000  |
+-----------------+-------+

    2. mysql处理线程缓存池大小¶

    mysql> show variables like '%thread_cache_size%';
+-------------------+-------+
| Variable_name     | Value |
+-------------------+-------+
| thread_cache_size | 512   |
+-------------------+-------+

    3. Mysql处理线程池情况¶

    mysql> show status like 'thread%';
+-------------------------+-------+
| Variable_name           | Value |
+-------------------------+-------+
| Threadpool_idle_threads | 0     |
| Threadpool_threads      | 0     |
| Threads_cached          | 490   | #当前缓存空闲处理线程数
| Threads_connected       | 25    | #当前已分配给连接的处理线程数
| Threads_created         | 53981 | #总计创建处理线程数
| Threads_running         | 3     | #当前正在执行处理线程数
+-------------------------+-------+

    4. 查看当前数据库的连接情况¶

    mysql> show processlist;

    5. 按照登陆用户+数据库+机器查看连接信息¶

    SELECT 
DB as database_name,
USER as login_user,
LEFT(HOST,POSITION(':' IN HOST)-1) AS login_ip,
count(1) as login_count
FROM information_schema.PROCESSLIST P 
GROUP BY DB,USER,LEFT(HOST,POSITION(':' IN HOST)-1);
